Share Data Viewer 
Author: Andrew Lau andrew.hk.lau@gmail.com
-------------------------------------------

A react.js application with ASP.NET Core 3.0 Web API backend.

Allows users to upload a CSV data file (samples can be found in the solution's "SampleData" directory) containing a number of shares and their unit price timeseries.

Built using Visual Studio 2019.

To run the solution, simply load the project in Visual Studio 2019 and run the project using IIS Express.
